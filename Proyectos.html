SICAD.NET

Sistema Integral de Controles y Adquicion de Datos, se utiliza para gestionar todos los accesos, reportes y configuraciones

Administración de Racks(Materia Prima)

Opcion para consultar el material que esta disponible en el almacen de materia prima, se puede filtrar por serie, parte o rack, se utiliza para la planeación del surtido(fifos), la información puede ser exportada a Excel para su análisis

Etiqueta Manual(Materia Prima)

Se utiliza para etiquetar material que no tiene etiqueta, esta dañada o esta mal identificado

Quebrado de Etiquetas(Materia Prima)

Se utiliza para etiquetar material que se separa en dos o más partes

Relaciones

En esta opción se registra el historial del material que sale del almacen por medio de baths manuales

Salidas a Corte

Aquí se validan y relacionan las series de materia prima y partes compradas que se utilizan para cada lote que se programa para producción

Salidas a Piel

Aquí se validan y relacionan las series de piel que se utilizan para cada lote que se programa para producción

Generar Lote Manual

Opción para validar estructuras de los numeros de parte y programar lotes para producción
